# 预审报告输出模板

本文档定义贷款预审报告的标准输出格式，所有 `review` 模式输出必须严格遵循此模板。

---

## 完整报告模板

```markdown
# 贷款申请预审意见书

**报告编号**：PRE-[日期]-[序号]
**预审日期**：YYYY-MM-DD
**申请人/企业**：[名称]
**贷款金额**：[金额] 万元
**贷款期限**：[期限] 个月
**贷款用途**：[用途描述]
**预审人（AI）**：本体预审助手 v1.0

---

> **[通过 / 有条件通过 / 不通过]**

|----------|------|------|
| 准入条件 | ✅通过 / ❌不符 / ⚠️待核实 | [简要说明] |
| 财务指标 | ✅通过 / ❌不符 / ⚠️待核实 | [简要说明] |
| 担保充足性 | ✅通过 / ❌不符 / ⚠️待核实 | [简要说明] |

## 四、推理路径摘要

> 每个结论的完整推理链，确保可追溯。

### 4.1 准入条件核验

```
申请人信用等级: AA
→ 规则 R-003（[文件名] 第X章第Y条）: 信用等级不低于 A 级
→ 结论: 满足准入条件 ✅
```

### 4.2 财务指标评估

```
资产负债率: 68%
→ 规则 R-001（[文件名] 第三章第十二条）: 资产负债率不超过 70%
→ 68% ≤ 70% → 满足 ✅

利息保障倍数: 1.8
→ 规则 R-007（[文件名] 第三章第十五条）: 利息保障倍数不低于 2.0
→ 1.8 < 2.0 → 不满足 ❌ → 触发风险标记
```

### 4.3 担保充足性评估

```
担保类型: 抵押
抵押率: 65%
→ 规则 R-012（[文件名] 第四章第三条）: 房产抵押率不超过 70%
→ 65% ≤ 70% → 满足 ✅
