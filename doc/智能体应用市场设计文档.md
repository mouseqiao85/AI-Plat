# 智能体应用市场设计文档

## 一、项目概述

### 1.1 项目背景
打造一个统一的智能体应用市场平台,整合金融智能体和通用技能中心,为用户提供一站式的AI能力发现、管理和使用入口。

### 1.2 设计目标
- 提供清晰的信息架构和导航体系
- 支持高效的搜索和筛选能力
- 展示丰富的智能体/技能信息
- 提升用户发现和使用效率

---

## 二、整体架构

### 2.1 页面结构
采用**左侧边栏 + 顶部导航 + 主内容区**的三栏布局:

```
┌─────────────┬──────────────────────────────────────┐
│             │  顶部导航栏                           │
│   侧边栏     ├──────────────────────────────────────┤
│   菜单       │                                       │
│             │  主内容区                              │
│             │                                       │
└─────────────┴──────────────────────────────────────┘
```

### 2.2 核心页面
1. **金融智能体应用市场** - 面向金融场景的专业智能体市场
2. **技能中心** - 面向通用场景的技能/工具市场

---

## 六、响应式设计

### 6.1 断点设置

- **桌面端**: ≥1200px (完整三栏布局)
- **平板端**: 768px - 1199px (侧边栏收起,主内容区自适应)
- **移动端**: <768px (侧边栏隐藏,汉堡菜单,单列卡片)

### 6.2 布局调整

**桌面端(≥1200px)**
- 侧边栏展开: 240px
- 金融智能体: 今日精选3列, 分类列表4列
- 技能中心: 2-3列

**平板端(768px - 1199px)**
- 侧边栏收起: 64px
- 金融智能体: 今日精选2列, 分类列表3列
- 技能中心: 2列

**移动端(<768px)**
- 侧边栏隐藏,汉堡菜单
- 金融智能体: 今日精选1列, 分类列表2列
- 技能中心: 1列
- 顶部导航简化

## 七、交互设计

### 7.1 搜索交互

**实时搜索建议**
- 输入时显示匹配结果
- 显示历史搜索记录
- 支持热门搜索推荐

**搜索结果页**
- 高亮显示搜索关键词
- 显示结果数量
- 支持在结果中筛选

### 7.2 筛选交互

**多选筛选**
- 分类标签支持多选
- 已选标签显示在顶部
- 支持快速清除筛选

**筛选结果**
- 实时更新,无需刷新
- 显示筛选后的数量
- URL同步筛选参数

### 7.3 卡片交互

**悬停效果**
- 阴影加深
- 轻微上移
- 显示更多操作按钮(收藏、分享)

**点击行为**
- 点击卡片主体: 跳转详情页
- 点击标签: 触发筛选
- 点击作者: 跳转作者主页
- 点击点赞/收藏: 快速操作

### 7.4 创建流程

**创建智能体/Skill**
- 点击"创建"按钮
- 弹出创建表单(模态框或新页面)
- 必填字段: 名称、描述、分类
- 可选字段: 图标、标签、权限设置
- 提交后跳转到编辑页面

## 九、技术实现建议

### 9.1 前端框架

- **框架**: React 18+ / Vue 3+
- **UI库**: Ant Design / Material-UI / 自研组件库
- **状态管理**: Redux Toolkit / Zustand / Pinia
- **路由**: React Router / Vue Router

### 9.2 性能优化

- 虚拟滚动(卡片列表)
- 图片懒加载
- 防抖/节流(搜索、筛选)
- 分页加载(无限滚动)
- 缓存策略(LocalStorage / IndexedDB)

### 9.3 可访问性

- 语义化HTML标签
- ARIA标签
- 键盘导航支持
- 屏幕阅读器支持
- 颜色对比度符合WCAG 2.1 AA标准
