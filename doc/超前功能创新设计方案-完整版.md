## 二、多智能体协作编排器 🤝

### 2.1 核心理念

> **构建智能体协作群落，解决单个智能体无法处理的复杂问题**

### 2.2 协作模式

#### 模式一：主从协作（Hierarchical）

```
        ┌─────────┐
        │ 主控Agent│
        └────┬────┘
             │
      ┌──────┼──────┐
      ↓      ↓      ↓
   ┌─────┐┌─────┐┌─────┐
   │Agent││Agent││Agent│
   │  A  ││  B  ││  C  │
   └─────┘└─────┘└─────┘
```

**适用场景**：
- 任务分解明确
- 需要统一决策
- 例如：银行贷款审批（主控+信贷+风控+合规）

#### 模式二：对等协作（Peer-to-Peer）

```
Agent A ←──────→ Agent B
   ↓                ↓
   ↓                ↓
Agent C ←──────→ Agent D
```

**适用场景**：
- 无中心控制
- 需要协商决策
- 例如：多方谈判、协同创作

#### 模式三：流水线协作（Pipeline）

```
Start → Agent A → Agent B → Agent C → End
          ↓          ↓          ↓
        [Queue]   [Queue]   [Queue]
```

**适用场景**：
- 顺序处理
- 流程标准化
- 例如：数据处理流水线

#### 模式四：竞争协作（Competitive）

```
         ┌→ Agent A →┐
         │           │
Start → Agent B → Consensus → End
         │           │
         └→ Agent C →┘
```

**适用场景**：
- 需要多样性观点
- 降低单一偏见
- 例如：投资决策、风险评估

### 2.3 可视化编排

## 五、智能体知识图谱自构建 🧠

### 5.1 核心理念

> **自动从文档中抽取知识，构建智能体专属知识图谱**

### 5.2 构建流程

```
文档上传
   ↓
实体抽取（NER）
   ├─ 人名、地名、机构名
   └─ 专业术语、概念
   ↓
关系抽取（RE）
   ├─ A影响B
   ├─ A属于B
   └─ A依赖B
   ↓
图谱构建
   ├─ 节点：实体
   └─ 边：关系
   ↓
向量化存储
   ├─ 图嵌入
   └─ 语义向量
   ↓
智能检索
   ├─ 图谱查询
   └─ 向量检索
```

### 5.3 技术实现

```python
class KnowledgeGraphBuilder:
    async def build_from_documents(self, docs):
        # 1. 实体抽取
        entities = await self.extract_entities(docs)
        
        # 2. 关系抽取
        relations = await self.extract_relations(docs, entities)
        
        # 3. 构建图谱
        graph = self.build_graph(entities, relations)
        
        # 4. 图嵌入
        embeddings = await self.embed_graph(graph)
        
        # 5. 存储
        await self.save_graph(graph, embeddings)
        
        return graph
    
    async def extract_entities(self, docs):
        """使用LLM抽取实体"""
        entities = []
        for doc in docs:
            # NER + LLM
            prompt = f"""
从以下文本中抽取所有重要实体：
- 人名、地名、机构名
- 专业术语、概念
- 产品、服务

文本：
{doc.content}

输出JSON格式：
{{
  "entities": [
    {{"name": "xxx", "type": "person", "importance": 0.9}}
  ]
}}
"""
            result = await self.llm.chat(prompt)
            entities.extend(result['entities'])
        
        return entities
    
    async def extract_relations(self, docs, entities):
        """抽取实体间关系"""
        relations = []
        
        for doc in docs:
            # 使用LLM抽取关系
            prompt = f"""
从以下文本中抽取实体间的关系：

文本：
{doc.content}

已知实体：
{[e['name'] for e in entities]}

输出JSON格式：
{{
  "relations": [
    {{"from": "A", "to": "B", "type": "影响", "strength": 0.8}}
  ]
}}
"""
            result = await self.llm.chat(prompt)
            relations.extend(result['relations'])
        
        return relations
```
